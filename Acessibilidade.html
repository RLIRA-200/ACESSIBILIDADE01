Navegação via teclado

Este site permite a navegação via teclado.

Os usuários que quiserem utilizar essa funcionalidade devem observar os seguintes comandos:

ALT + 1:  página inicial

ALT + 2: Redireciona para o conteúdo principal

ALT + 3: Redireciona o usuário para o campo de busca do portal

Observações:

Orientamos o usuário a utilizar o Browser Google Chrome para a navegação via teclado.

Mozilla Firefox:  pressionar Alt + Shift + tecla de atalho correspondente, conforme menu acima.

 

Acessibilidade para pessoas com deficiência auditiva

Este site é acessível em Libras - Língua Brasileira de Sinais.

A acessibilidade para as pessoas com deficiência auditiva depende do download do Suíte Vlibras, que é um conjunto de ferramentas computacionais de código aberto, disponibilizado no Portal do Software Público Brasileiro (SPB).

Essa solução possibilita traduzir conteúdos digitais (texto, áudio e vídeo) para a Língua Brasileira de Sinais - LIBRAS, tornando computadores, dispositivos móveis e plataformas Web acessíveis para pessoas surdas.

 

Acessibilidade para pessoas com deficiência visual

Este site é acessível a pessoas com deficiência visual.

A acessibilidade para essas pessoas depende da seleção do botão de alto-contraste existente na parte de cima do site, à direita.
